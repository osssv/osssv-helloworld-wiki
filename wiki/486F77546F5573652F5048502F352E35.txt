-CentOS installation (You can refer [[Overall/CentOS]])

*Install&Setup [#rff91024]
:Step.1|
Execute the following command.

 $ sudo yum install php55*

php5.5 requires httpd later than 2.4, so if you haven't installed httpd yet, it will install httpd2.4 automatically.
In case you have already had older version of httpd, it may be failed.
If you possible, clean install is easier.
